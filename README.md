# Gestor_de_Tareas-PrimerRepo-
Este es mi primer repositorio.

El proyecto será muy básico, es solo para practicar mi uso de MySQL, Python, HTML y CSS. En cuanto a Python, se usarán librerías como flask, mysql, dotenv, etc(para mas detalles, leer "requirements.txt"). La idea es hacer un gestor de tareas de un usuario. Por el momento se espera que la aplicación tenga registro de usuarios; creacion, edición y eliminación de tareas; ver tareas por prioridad en base a fechas y prioridades.
